# two-factor-auth
Two-factor auth for SSH + LDAP

Dependencies:
- libboost-dev

---

## Manager
### Actions (rw)
- Add two-factor attrs
- Remove two-factor attrs
- (Re)generate TOTP token
- (Un)ban account

- :soon: Generate QR code for TOTP token
- :soon: Show accounts with active sessions
- :soon: Show banned accounts
